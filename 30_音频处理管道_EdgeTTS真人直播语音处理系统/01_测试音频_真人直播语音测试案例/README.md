## 🎧 试听建议

1. **按场景顺序试听**: 原始 → 新闻 → 教学 → 聊天 → 娱乐 → 游戏
2. **对比试听**: 感受不同场景的语速、音高、背景音差异
3. **重点试听**: 语速变化、音高变化、背景音效、事件音效
